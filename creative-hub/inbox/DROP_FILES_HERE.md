# Inbox — Drop Zone

Drop your Higgsfield exports here. GitHub Actions will automatically:

1. Detect file type (video, image, design)
2. Rename with timestamp
3. Move to the correct folder in `assets/`
4. Commit everything — no manual work needed

## How to upload

**From your phone (easiest):**
1. Export/save from Higgsfield app
2. Go to github.com/jaystruckin/TKLINK
3. Navigate to `creative-hub/inbox/`
4. Tap "Add file" → "Upload files"
5. Pick your exported file(s)
6. Tap "Commit changes" — done!

**From desktop:**
- Same steps, or drag & drop files right into this folder on GitHub.com

Files are auto-sorted into:
- `assets/videos/` — mp4, mov, webm, etc.
- `assets/images/` — jpg, png, gif, webp, etc.
- `assets/designs/` — psd, pdf, ai, etc.
